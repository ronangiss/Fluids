Introduction

   This repository is a place for me and you to review core fluid physics concepts. You still need a textbook to learn the material in the first place, but if you're like me and you've learned it before but can't remember it now, this website is for you.

   This website was inspired by Paul's Online Math Notes and seeks to make fluid physics education more accesible through the use of interactive features available in javascript and not in a traditional textbook hardcopy or pdf. The mass conservation page is the most populated page at the moment.
